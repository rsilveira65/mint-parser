# MINT Parser
## About
Script to parse feed from GitLab, TeamWork and Hipchat.

## Configuration
To configure the application, set the follow parameters:
```bash
.
└── mint-paser
    └── config
        └── parameters.yml
```

## Run
In your console, run:

```bash
composer install
```

```bash
php mint-parser.php run
```
